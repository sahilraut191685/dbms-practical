Write a named PL/SQL block using a Stored Procedure to categorize students based on
their total marks.
Consider the following tables:
o Stud_Marks (name, total_marks)
o Result (roll, name, class)
Create a stored procedure named proc_Grade to assign a class to students based on their
marks:
 Marks 990 to 1500 → Distinction
 Marks 900 to 989 → First Class
 Marks 825 to 899 → Higher Second Class
 Create the Stored Procedure (proc_Grade) to categorize students
 Insert the categorized data into the Result table
 Write a PL/SQL block to call the procedure

```sql
-- CREATE TABLES
CREATE TABLE Stud_Marks (
    roll INT PRIMARY KEY,
    name VARCHAR2(50),
    total_marks INT
);

CREATE TABLE Result (
    roll INT,
    name VARCHAR2(50),
    class VARCHAR2(50)
);

-- INSERT SAMPLE DATA
INSERT INTO Stud_Marks VALUES (1, 'Sahil', 1200);
INSERT INTO Stud_Marks VALUES (2, 'Rahul', 950);
INSERT INTO Stud_Marks VALUES (3, 'Priya', 870);
INSERT INTO Stud_Marks VALUES (4, 'Amit', 800);
INSERT INTO Stud_Marks VALUES (5, 'Neha', 1000);

-- STORED PROCEDURE
CREATE OR REPLACE PROCEDURE proc_Grade (
    p_roll IN INT,
    p_name IN VARCHAR2,
    p_marks IN INT
)
IS
    v_class VARCHAR2(50);
BEGIN
    IF p_marks BETWEEN 990 AND 1500 THEN
        v_class := 'Distinction';
    ELSIF p_marks BETWEEN 900 AND 989 THEN
        v_class := 'First Class';
    ELSIF p_marks BETWEEN 825 AND 899 THEN
        v_class := 'Higher Second Class';
    ELSE
        v_class := 'Pass';
    END IF;

    INSERT INTO Result VALUES (p_roll, p_name, v_class);
END;
/

-- PL/SQL BLOCK TO CALL PROCEDURE
BEGIN
    FOR rec IN (SELECT roll, name, total_marks FROM Stud_Marks) LOOP
        proc_Grade(rec.roll, rec.name, rec.total_marks);
    END LOOP;
END;
/

-- VIEW RESULT
SELECT * FROM Result;
```
